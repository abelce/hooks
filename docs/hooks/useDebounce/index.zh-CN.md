# useDebounce

创建一个 debounced 函数，在延迟`wait`毫秒后，返回输入的的`value`

## 代码演示

<code src="let-hooks/useDebounce/demos/base.tsx" title="基本用法"></code>

## 参数

| 参数  | 描述           | 类型   | 默认值 |
| ----- | -------------- | ------ | ------ |
| value | 防抖返回的值   | any    | --     |
| wait  | 等待时间(毫秒) | number | 0      |
